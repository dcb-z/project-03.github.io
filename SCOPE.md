# Project Scope Statement: Photography Portfolio Website

## Project Purpose
The purpose of this project is to create a personal photography portfolio website that showcases photos I have taken. This website will help organize my work in one place and present my photography in a professional and visually appealing way.

## Project Description
The final project will be a website that includes a homepage, a photography gallery, an about section, and a contact section. The gallery will display selected photos from my photography work, such as portraits, events, nature, or creative images. The website should be easy to navigate and visually clean so the photos remain the main focus.

## Project Objectives
- Create a professional photography portfolio website.
- Showcase selected photos in an organized gallery.
- Include a short introduction about me and my photography style.
- Provide a simple way for viewers to contact me.
- Design the website with a clean, creative, and user-friendly layout.

## Deliverables
- A completed photography portfolio website.
- A homepage introducing the portfolio.
- A photo gallery page or section.
- An about section describing my photography interests.
- A contact section with basic contact information.
- A polished design using HTML and CSS.

## Project Requirements
- The website must be visually organized and easy to use.
- Photos should be displayed clearly and professionally.
- The design should support the purpose of the site instead of distracting from the images.
- The site should include working navigation links.
- The final files should be submitted according to the course instructions.

## Project Boundaries
This project will focus on creating a basic photography portfolio website. It will not include online booking, payment features, client logins, or advanced photo editing tools.

## Assumptions
- I will use my own photography images for the website.
- The website will be created using the tools allowed in class.
- The final project will build on skills learned in previous assignments.
- The website will be designed for a general audience interested in viewing my photography.

## Constraints
### Time
The project must be completed within the course timeline.

### Quality
The website should be polished, readable, and visually appealing.

### Budget
There is no expected budget because I will use my own photos and free web design tools.

## Success Criteria
The project will be successful if the final website clearly showcases my photography, is easy to navigate, has a professional appearance, and meets all course requirements.
