# Project Plan

## Project Overview
The goal of this project was to create a professional photography portfolio website called “Memories Captured by Donna Baca.” The website was designed to showcase photography work while applying web development skills learned throughout the semester using HTML, CSS, GitHub, and GitHub Pages.

The project includes multiple webpages, organized folders for images and videos, external CSS styling, navigation links, and embedded multimedia content.

---

# TAME Risk Analysis

## Technical Risks
One major risk involved learning GitHub and GitHub Pages for the first time. There was a possibility of broken file paths, publishing errors, missing images, or pages not displaying correctly online.

### Solution
To reduce this risk, I tested my website frequently after updates and reviewed my file structure carefully. I also organized images and videos into separate folders to improve repository organization.

---

## Artistic Risks
Another risk was creating a design that looked visually appealing and professional while still being functional and easy to navigate.

### Solution
Peer feedback was used to improve the site by adding in a high resolution image gallery. Multiple revisions were made to improve the overall aesthetic and user experience.

---

## Management Risks
Time management became a challenge because the project involved both technical coding and creative design work. Small coding mistakes often took additional time to troubleshoot.

### Solution
The project was divided into smaller tasks completed over several days, including webpage setup, CSS styling, image uploads, navigation links, and documentation.

---

## Environmental Risks
Technical issues such as GitHub Pages deployment delays, browser caching, and incorrect media formatting created obstacles during development.

### Solution
Testing was completed across multiple browsers and devices. Files were renamed and reorganized when necessary to ensure proper functionality.
